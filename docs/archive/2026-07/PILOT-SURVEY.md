# NikoLearn — პილოტის უკუკავშირი (კლასი, 20–25 ბავშვი)

მიზანი: **ინსაიდი** — რა მუშაობს, სად ვართ სუსტი, დაბრუნდება თუ არა ბავშვი. 20–25 ოჯახზე
საუკეთესო სიგნალი ხარისხობრივია (survey + რამდენიმე საუბარი), არა dashboard.

## ვადები
- დღე 0: მშობლებს გადაეცემათ ბმული + `PARENT-GUIDE.md` (აუცილებლად „Add to Home Screen").
- დღე 7: გაეგზავნოს ქვემოთ მოცემული 5-კითხვიანი survey (Google Form).
- დღე 7–10: 2–3 მშობელთან მოკლე საუბარი (10 წთ) სიღრმისთვის.

## მშობლის survey (Google Form — 5 კითხვა)
1. ბავშვი თავისით დაბრუნდა აპში? (ყოველდღე / რამდენჯერმე / ერთხელ / არა)
2. რომელი ნაწილი ყველაზე მოეწონა? (თვლა / ანბანი / სიტყვები / მათემატიკა / ფრაზები)
3. იყო რამე, რაც გააღიზიანა ან ვერ გაიგო? (ღია პასუხი)
4. რამდენ ხანს თამაშობდა ერთ ჯერზე? (—5 წთ / 5–15 წთ / 15+ წთ)
5. გააგრძელებდი გამოყენებას? რას დაამატებდი? (ღია პასუხი + 1–5 შეფასება)

## მასწავლებელთან/მშობელთან საუბრის კითხვები (qual)
- სად გაუჭირდა ბავშვს ისე, რომ თქვენ უნდა დახმარებოდით?
- რა მომენტში გაუხარდა ან „აჩვენა"?
- ენდობით აპს უსაფრთხოების მხრივ? რა გაგიჩენდათ მეტ ნდობას?
- ეს თქვენი შვილისთვის სასარგებლოა? რატომ?

## რას ვუყურებთ (success სიგნალები)
- **დაბრუნება** (#1) — ყველაზე მნიშვნელოვანი: ბავშვი თავისით ბრუნდება?
- **სითბო/ნდობა** — „📣 მაჩვენე მშობელს" გამოიყენეს? მშობელს მოეწონა?
- **სად ვართ სუსტი** — კითხვა #3 + dashboard-ის „დასახმარებელია".
