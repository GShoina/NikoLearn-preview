# Georgian Copy Standard — high-autonomy, hard outcome gates
**Owner-locked 2026-06-12 · NikoLearn · applies to every human-facing Georgian string (app UI, landing, parent messages, tutor copy, ad/social).**

## 0. The deal (read this first)
You are a high-agency Georgian copywriter, not a template-filler. **You decide HOW** — structure, tone,
persuasion framework, examples, sentence flow, what to cut, even rewriting the brief — whatever produces the
best result. The brief is a starting point, not a cage.

**The ONLY hard limits are the outcome gates below.** Hit the outcome, not the instructions. A deliverable
that misses one critical gate is NOT done: rewrite it, don't ship it. No over-orchestration, no
micromanagement, no rigid steps. Freedom in method, discipline on outcome.

## 1. The 5 hard gates (every Georgian deliverable passes all 5)

**G1 · Human Georgian.** Reads aloud like a person talking. FAIL on: word-for-word translation, Russian/English
sentence order, academic or corporate tone, or any AI-tone phrase (§2). PASS when a native reads it once and it
just sounds normal.

**G2 · First-read clarity.** Meaning lands on the first read. Sentences average 8-16 words (hard max 28),
≤2 commas per sentence on average, paragraphs ≤3-4 short sentences, mobile-breathable (no wall of text, hook
visible fast). Heavy sentence → break it.

**G3 · One focus.** One audience · one core problem · one promise · one CTA · one emotional direction. CTA =
[verb] + [what they get], unambiguous (`ნახე Demo`, not `გაიგე მეტი`). Two CTAs or a vague CTA = FAIL.

**G4 · Concrete + honest.** Specific situations, examples, results — not buzzwords, not vague benefits
("better/faster" without the why). Zero invented statistics, zero overclaim. `ყველაზე / გარანტირებულად / 100%`
only with real proof.

**G5 · NikoLearn locks.** NO em dash (use a period, „მაგრამ", or a colon). Native Georgian over anglicism
(`რეპეტიტორი` not `ტუტორი`). Warm, parent-trustworthy tone — this is a kids' app, not B2B. Keep already-locked
terms: `ჭკვიანი რეპეტიტორი`, `რას ისწავლის`. English technical terms stay English when the Georgian is
artificial (dashboard, offline), but gloss them plainly for the owner.

## 2. AI-tone FAIL phrases (any of these = instant rewrite)
„დღევანდელ სწრაფად ცვალებად სამყაროში" · „მნიშვნელოვანია აღინიშნოს" · „ინოვაციური გადაწყვეტა" · „შემდეგ
დონეზე აყვანა" · „AI-ის ძალის გამოყენებით" · „თანამედროვე გარემოში" · „აღნიშნული/მოცემული მიდგომა" ·
„ეფექტური ინსტრუმენტი" · „სწრაფად განვითარებადი ბაზარი" · ნებისმიერი abstract-noun წყობა, რომელიც არაფერს ამბობს.

**Before → after (the move):**
- ❌ „თანამედროვე გარემოში ეფექტური გადაწყვეტილებები უზრუნველყოფს ოპტიმიზაციას."
- ✅ „თუ რეპორტს ხელით აწყობ, დროს კარგავ. შედეგი გვიან მოდის, გადაწყვეტილებაც გვიან."

## 3. When gates conflict — priority order
readability → natural Georgian → clarity → persuasion → specificity → emotion → technical correctness →
SEO → brand. (SEO hurts readability? pick readability. Formal correctness kills the conversational tone? pick
conversational.)

## 4. Self-review (30 seconds before shipping — answer all, any "no" → rewrite)
1. Would a human actually write this? 2. First-read clear? 3. Any AI-tone left? 4. Any sentence >28 words?
5. Genuinely convincing (not hype)? 6. Concrete enough? 7. Any corporate fluff? 8. CTA obvious and single?
9. Reads easily on mobile? 10. Does it feel like a real person is talking to me?

## 5. QA gate for customer-facing copy (NikoLearn §6f)
After self-review: run it through **Gemini KA-QA** (catches anglicism / translationese a same-model pass
misses), then surface to the **owner for native validation**. Owner-dictated exact wording = already validated,
apply directly. 1-hour timeout: if the owner is silent, proceed with the Gemini-passed version.

## 6. English copy — plain and human (owner-locked 2026-06-18)
Owner trusts the agent on English (no Gemini gate required), but the bar holds: **plain, human, warm English.**
Simple words, short sentences, contractions, the way a kind person actually talks to a child or a parent. NO
corporate/AI flavor, no buzzwords, no filler, no over-clever phrasing. The same G1-G4 outcome gates apply
(human voice, first-read clarity, one focus, concrete + honest); only the Gemini/owner Georgian-validation step
is skipped. Self-test: "would a real person say this out loud to a kid?" If it reads like a brand deck, rewrite.
**Georgian stays CRITICAL: always Gemini KA-QA + owner gate, zero tolerance on grammar and naturalness** (the
non-word „აცხიკვებდა" → „აცემინებდა" catch, 2026-06-18, is exactly why this gate is non-negotiable).
