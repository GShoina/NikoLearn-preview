# NikoLearn — Math Ideas Library (owner reference, 2026-06-13)

Source: owner shared worksheets/photos from Nanobashvili 1st-grade book + „1500 ტესტი" collection
(Goshkheteliani/Kostava, grades 6-9 — too advanced, methodology only). Take IDEAS + METHODOLOGY,
improvise, age-fit to NikoLearn (3-9). **TUTOR FIRST: every type must teach + spark interest, not just drill.**

## Idea types seen (status)
1. **Multi-step (3-term) arithmetic** — `9 + 9 − 8`, `15 − 5 − 3`. ✅ DONE v1.145 (op:multi for 7+, step-by-step tutor hint).
2. **Word problems, varied real-life scenarios** — found / combine / two-boxes-different / give / eat / birds /
   comparison-difference / shopping-money / equal-sharing-division. ✅ DONE v1.144 (10 scenarios, food↔eat vs object↔give).
3. **Patterns, varied + blank in any position** — varied step / skip 5-10 / decreasing / doubling ×2; blank middle or last.
   ✅ DONE v1.142+1.144.
4. **Missing-number, blank EITHER side, multi-step** — `10 − 7 + 4 = □ − 2`, `10 + 5 − □ = 7`, `□ − 3 = 12 − 4 − 3`.
   math-miss exists but is simple (`? op y = res`). ▶ ENHANCE: blank either side, multi-term, balance both sides.
5. **Expression comparison** — `9 + 1 ▢ 9`, `11 + 3 ▢ 11` (compute then >,<,=). `compare` exists for bare a?b.
   ▶ ENHANCE: sometimes compare an EXPRESSION to a number.
6. **★ Picture-substitution puzzles (OWNER FAVORITE, sent 4×, most tutor-first)** — `🏉 = 🌲 + 🚗`, `🏉=9₾, 🚗=4₾, 🌲=?`;
   jars+weights on a balance; `🧸+🧸=13₾, 🧸+🚗=?`. Reasoning / pre-algebra with pictures. Niko explains the logic
   („ორი ბურთი 9-ია, ერთი 4, მეორე 5"). ▶▶ BUILD NEXT as its own mode. Simple version = some values given, solve the
   missing pictured item (= subtraction/division with pictures); avoid full 2-unknown systems for young.
7. **Visual / pictorial arithmetic (concrete-first)** — count circle/dot groups (white/black/hatched) → write the
   equation `4 + 3 + 1 = 8`. Shows quantity, not just digits. ▶ ROADMAP: pair a visual object-group with the sum
   (great for 5-7, tutor-first „see why").
8. **Number-line / ordering** (short↔tall, sequence) — different interaction (order/drag). Lower priority, defer.
