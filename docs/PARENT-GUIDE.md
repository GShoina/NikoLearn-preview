## როგორ დავიწყოთ (2 წუთი)
1. **გახსენი ბმული** ტელეფონში: `https://gshoina.github.io/NikoLearn/NikoLearn Phase 1.html`
2. **⭐ აუცილებლად: „Add to Home Screen" / „მთავარ ეკრანზე დამატება".**
   - iPhone: Safari → გაზიარების ღილაკი → „Add to Home Screen".
   - Android: Chrome → ☰ → „Install app" / „Add to Home screen".
   - **რატომ მნიშვნელოვანია:** ასე აპი ნამდვილი აპივით იხსნება, ინტერნეტის გარეშე მუშაობს და
     **ბავშვის პროგრესი არ იკარგება.** (თუ მთავარ ეკრანზე არ დაამატებ, iPhone-მა შეიძლება
     მონაცემი ~1 კვირაში წაშალოს.)
3. **შესვლა:** პაროლი `12345`.
4. **შექმენი ბავშვის პროფაილი:** სახელი, ასაკი, ფერი და — **„კითხვა იცის?"** (თუ ჯერ ვერ კითხულობს,
   აირჩიე „ჯერ არა 🖼️" — აპი სურათებით და ხმით ასწავლის, კითხვის გარეშე).
